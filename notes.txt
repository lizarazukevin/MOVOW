Prerequisites:
- Install anaconda or mini conda
- Install VSCode
- Install pip installer

Creating the djangoenv
1. Create a conda environment using: conda create --name movow python==3.10
2. Pip install django
3. Create project: django-admin startproject movowDjango 
4. Create application: django-admin startapp home
5. Fill site with URL routing
4. Observe site: python manage.py runserver
